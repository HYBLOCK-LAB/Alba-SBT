# Alba-SBT — 블록체인 기반 알바 경력 증명 시스템

> 알바생의 출퇴근 데이터를 블록체인에 자동 기록하고, 조작 불가능한 경력 배지(SBT)를 발급합니다.

---

## 배경

알바생의 경력 증명 방식에는 세 가지 구조적 문제가 있습니다.

| 문제 | 내용 |
|------|------|
| **파편화** | 고용보험, 통장 입금 내역, 매장 기록이 흩어져 있어 통합 관리 불가 |
| **휘발성** | 4대 보험 미가입 알바는 사장님이 기억하지 않으면 기록 자체가 사라짐 |
| **수동성** | 경력 증명서를 매번 직접 요청·발급받아야 하는 번거로움 |

또한 채용 플랫폼에서는 허위 경력 작성과 경력 부풀리기로 인해 신뢰 기반이 무너지고 있습니다.

---

## 솔루션

**EAS(Ethereum Attestation Service) + SBT(Soulbound Token)** 를 결합한 자동 경력 증명 시스템입니다.

- **데이터 기반 수집**: QR 코드와 GPS로 출퇴근을 객관적으로 검증하여 자동 기록
- **자동 발행**: 조건 달성 시 EAS가 자동 발행되며, 사장님 개입이 필요 없음
- **온체인 불변 기록**: 모든 증명이 블록체인에 영구 저장되어 조작 불가, EAS Explorer로 누구나 검증 가능

---

## 주요 기능

### 알바생 (worker)

- **QR / GPS 출퇴근**: 매장 반경 50m 이내 GPS 또는 사장님이 발급한 10초 유효 QR로 출퇴근 기록
- **경력 프로필 대시보드**: 보유 SBT 배지 시각화, 레벨업 프로그레스 바 제공
- **포트폴리오 QR 공유**: 지갑 주소 기반 QR 생성 → 채용처에서 즉시 경력 검증 가능
- **추가 근무 신청**: 사장님이 등록한 추가 근무 요청 확인 및 신청 (스케줄 충돌 자동 감지)

### 사장님 (manager)

- **매장 관리**: 매장 등록, 6자리 매장코드 자동 생성, 알바생 승인·삭제
- **근태 관리**: 월간 캘린더 및 금일 실시간 출근 현황 확인
- **출퇴근 QR 생성**: [QR 생성] 버튼으로 10초간 유효한 당일 전용 QR 발급
- **승급 승인**: 조건 충족 알바생을 목록에서 선택 → EIP-712 서명 → SBT 자동 발급
- **지원자 경력 검증**: 포트폴리오 QR 스캔 → EAS/SBT 신뢰도 리포트 즉시 조회

## 블록체인 설계

### EAS — 팩트 증명 자동 발행

| EAS 유형 | 이름 | 발급 트리거 |
|----------|------|------------|
| `EAS_EXP_TIME` | 업종 경력 증명 | 동일 매장 6개월 근무 달성 (매장별 독립 판정) |
| `EAS_FAITH_ATT` | 성실성 증명 | 90일 구간 내 결근 0회 + 지각 2회 이하 |
| `EAS_WORK_COMP` | 근무 완수 증명 | 성실성 조건 충족 + 조기 퇴근 0회 |
| `EAS_EXTRA_ACC` | 추가 근무 수락 증명 | 추가 근무 수락 누적 10회마다 1개 발급 |

### SBT — 레벨 배지 시스템

| 티어 | 레벨 | 최소 달성 시기 |
|------|------|-------------|
| 🛡️ Beginner | Lv.1 ~ Lv.4 | 가입 즉시 ~ 약 18개월 |
| ⚔️ Intermediate | Lv.5 ~ Lv.7 | 약 2년 ~ 30개월 |
| 🧙 Expert | Lv.8 ~ Lv.9 | 약 3년 ~ 42개월 |
| 👑 Master | Lv.10 | 약 4년+ |

SBT 승급은 **EIP-712 기반 Multi-Signature** 방식으로 처리됩니다. 사장님의 오프체인 서명과 플랫폼 서명을 결합해 가스비 없이 스마트 컨트랙트를 통해 SBT를 발급합니다.

## 기술 스택

| 영역 | 기술 |
|------|------|
| **Mobile** | React Native (Expo) — iOS / Android 동시 지원 |
| **Backend** | NestJS + Supabase (PostgreSQL) + Supabase Edge Functions |
| **Blockchain** | Ethereum Sepolia + EAS SDK + ERC-5192 (SBT) + EIP-712 Multisig |
| **Wallet** | MetaMask / WalletConnect (SIWE 인증) |
| **Storage** | Supabase Storage (SBT 메타데이터, 배지 이미지) |

## 시작하기

### 환경 변수 설정

`.env.example`을 복사해 `.env`를 생성하고 각 값을 채워주세요.

```bash
cp .env.example .env
```

### DB 스키마 적용

```bash
# Supabase CLI 사용
supabase db push

# 또는 schema.sql 직접 실행
psql -h <host> -U <user> -d <db> -f schema.sql
```

### 앱 실행

```bash
# 의존성 설치
npm install

# Expo 개발 서버 시작
npx expo start
```
